# 毛球日记 (Maoqiu Diary) VIP 专属功能规划

基于当前项目的业务逻辑（治愈、手绘风格、AI助手、多媒体记录、分享等），整理以下 VIP 专属功能规划：

## 1. AI 助手与智能分析 (推荐，可有效控制 API 成本)
*   **AI 树洞畅聊**：限制普通用户每天只能和“毛球”对话 5-10 次，VIP 用户可以无限次畅聊。
*   **AI 智能周报/月报**：VIP 用户可解锁由 AI 基于近期的日记内容和心情追踪，自动生成的专属情感分析和治愈信件。
*   **AI 帮写与润色**：为 VIP 提供“一键润色”、“灵感扩写”或“根据心情自动配文”的功能。

## 2. 分享卡片与数据留存 (提升社交展示与数据安全)
*   **免水印分享卡片**：目前的分享卡片底部带有 `—— 来自毛球日记 ——` 的标识，VIP 用户可以获得关闭水印的特权。
*   **专属分享背景/模板**：普通用户只能使用基础的场景模板，VIP 可以解锁更多高级手绘风格的分享模板和精美背景图。
*   **数据导出 (PDF/Markdown)**：允许 VIP 用户将某个日记本或全部日记导出为精美的 PDF 或 Markdown 文件，满足永久本地保存的需求。

## 3. 多媒体与云存储特权 (平衡服务器成本)
*   **高清原图与视频记录**：普通用户上传图片可以进行压缩且限制九宫格；VIP 用户则可以上传高清原图，或者解锁短视频日记。
*   **云存储空间扩容**：普通用户限制一定容量（如 500MB），VIP 提供更大的云同步空间（如 10GB）。

## 4. 个性化装扮 (提供情绪价值)
*   **专属毛球形象与互动**：VIP 可以解锁特殊皮肤的毛球（如节日限定、动态服装款），以及更丰富的专属互动动画。
*   **应用主题与专属字体**：VIP 可自定义应用的主题色，并解锁精美的手写体，甚至允许自定义 App 桌面图标。

## 5. 高级管理与隐私保护
*   **无限日记本**：限制普通用户最多只能创建 3 个日记本，VIP 可创建无限个并进行高级分类。
*   **高级隐私锁 (Face ID/指纹)**：为特定日记本或整个应用开启生物识别解锁，保护用户的私密日记。

## 6. 社区专属身份
*   **VIP 尊贵标识**：在动态列表和评论区，VIP 用户昵称旁可显示专属徽章标识，或者拥有特殊的卡片样式。
