Product Values

产品特点

毛球日记希望做的不只是记录工具，而是一个能让人安心停留、愿意反复回来的情绪空间。

这部分强化了“产品调性”而不是只堆功能，能更好地支撑官网气质和上架展示。

温柔治愈的记录体验

  • 围绕“生活、情绪、回忆”设计整体体验，适合长期书写与回看。
  • 支持分类、照片墙、时间轴等多种回顾方式，帮助用户重新看见自己的生活轨迹。
  • 提供主题模式、通知提醒和应用锁等能力，兼顾仪式感与私密性。

真实、克制的社交关系

  • 用户可以选择在圈子中分享公开内容，也可以把更多日常留在自己的私人空间。
  • 通过关注、主页浏览与互动建立连接，但仍以“记录自己”而非“被动表演”为核心。
  • 共享日记本让亲密关系中的共同记录更自然、更有温度。
