# 毛球日记 - 快速开始指南

## ⏱️ 5 分钟快速上手

### 步骤 1: 安装依赖 (1 分钟)

```bash
# 进入项目目录
cd maoqiu-diary-app

# 安装依赖
npm install
```

### 步骤 2: 配置环境变量 (1 分钟)

```bash
# 复制环境变量示例文件
cp .env.example .env

# 编辑 .env 文件
# 确保 TCB_ENV_ID 已配置为你的腾讯云环境 ID
```

编辑 `.env`:
```bash
TCB_ENV_ID=maoqiu-diary-app-2fpzvwp2e01dbaf
TCB_REGION=ap-guangzhou
```

### 步骤 3: 启动开发服务器 (1 分钟)

```bash
# 启动 Expo
npm start
```

屏幕上会出现二维码：
- **真机测试**: 使用 Expo Go App 扫码
- **iOS 模拟器**: 按 `i` 键
- **Android 模拟器**: 按 `a` 键

### 步骤 4: 体验应用 (2 分钟)

1. **登录/注册**
   - 输入手机号
   - 点击"获取验证码"
   - 输入验证码（开发环境：123456）
   - 完成登录

2. **创建日记**
   - 点击首页右下角"+"按钮
   - 填写标题
   - 选择日期、心情、天气
   - 点击"保存"

3. **查看日记**
   - 在首页时间轴查看已创建的日记
   - 点击日记查看详情

## ❓ 常见问题

### Q: 验证码收不到？

**A**: 开发环境使用测试码 `123456`，生产环境需要配置短信服务。

### Q: 数据没有保存到云端？

**A**: 
1. 检查 `.env` 文件中的 TCB 环境 ID 是否正确
2. 确认云函数已部署
3. 查看网络请求日志

### Q: 如何修改主题颜色？

**A**: 编辑 `src/config/handDrawnTheme.ts` 文件。

### Q: 如何添加新的萌宠角色？

**A**: 编辑 `src/config/mascot.ts` 文件，添加新的萌宠配置。
